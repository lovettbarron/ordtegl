- Border-bottom for visual separation
- Padding: $spacing-sm vertical
- Navigation: horizontal flex layout, space-between
- Logo/title: larger font, no text-decoration
- Nav links: $color-text-primary, hover: $color-accent
- Mobile: stack navigation vertically below 768px breakpoint

- Padding: $spacing-lg vertical for breathing room
- Min-height to push footer down

Footer styles:
- Background: $color-surface (cream)
- Border-top for visual separation
- Padding: $spacing-md vertical
- Text: $color-text-secondary
- Links: $color-accent

Mobile-first approach:
- Base styles for mobile
- @media (min-width: 768px) for desktop overrides

Update `assets/css/main.scss` to import layout:
